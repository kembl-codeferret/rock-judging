# Rock Judging program for the Kembl Rock Awards

I made this program to aid in judging the annual rock contest that i hold every year. 
If you find this and want to take part the discord server is https://discord.gg/jmmS3JpR9g

If you are running the program (for some reason) you will need pyqt6, gdown, and pillow (all of which you can pip install)
